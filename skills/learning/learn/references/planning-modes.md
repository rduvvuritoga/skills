# Planning Modes

Use only the section for the selected mode.

## `80-20`

Build a 20-hour learning plan as ten two-hour blocks focused on the smallest set of concepts that unlock useful practice.

For every block include:

- What to learn.
- One high-signal resource identified precisely enough to find.
- One practice task.
- A 15-minute check that demonstrates understanding.

Order blocks by dependency and practical payoff. Completion requires all ten blocks, no duplicate objectives, and a check whose result distinguishes understanding from passive exposure.

## `onepager`

Create a compact page the learner can review in about five minutes. Include:

- Core concepts and their relationships.
- Common mistakes.
- One real example for each major concept.
- One memorable mental model.
- A short self-check.

Use dense but readable headings and bullets. Completion requires the topic's essential relationships to fit in one coherent view rather than a shortened tutorial.

## `ladder`

Map five behavioral levels from “I know nothing” to “I could teach this.” For each level include:

- What the learner can do.
- What to study.
- A realistic time range.
- A practice result that proves readiness for the next level.

Completion requires observable boundaries; credentials, course completion, and time spent do not prove a level.

## `mistakes`

Identify the five beginner traps most likely to create wasted effort, false confidence, or harmful habits. For each explain:

- Why the trap is attractive.
- The cost it creates.
- The better behavior.
- A simple rule or check that prevents it.

Completion requires topic-specific traps rather than generic advice such as “practice more.”
