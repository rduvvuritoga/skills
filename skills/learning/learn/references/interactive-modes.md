# Interactive Modes

Use only the section for the selected mode. End the current response at the learner prompt; subsequent steps belong to later turns.

## `newbie`

Explain the topic with simple words, a concrete everyday comparison, and no unexplained jargon. Then ask three short questions that test the core mental model.

After the learner answers, identify what is correct, re-explain only the weak parts, and ask a smaller follow-up check. Completion occurs when the learner answers the core questions accurately in their own words.

## `teachback`

Give the smallest explanation needed for a teachback, then ask the learner to explain it in their own words.

After each response:

1. Identify what is accurate.
2. Identify what is missing or distorted.
3. Correct only those parts.
4. Ask for the next teachback.

Completion occurs when the learner can explain the topic clearly to a friend without notes.

## `test`

Prepare ten practical questions that test transfer and reasoning rather than recall. Ask one question per turn.

After each answer, explain what it reveals, correct the reasoning when needed, update the running score or competency map, and ask the next question. Completion requires all ten answers plus a final summary of strengths and next practice priorities.

## `scenarios`

Create five realistic situations in which the learner would use the topic. Walk through the first scenario step by step, then ask the learner to attempt the second.

For scenarios two through five, let the learner finish one attempt before giving feedback. Completion requires all five scenarios and a final statement of the decision pattern that transfers across them.
