# Bridge Mode

Teach a new topic by mapping it to something the learner already understands.

Parse:

```text
/learn bridge [familiar topic] -> [new topic]
```

Proceed without the arrow when both topics are still clear. Otherwise ask for the familiar topic and target topic.

Explain:

- The shared structure or behavior.
- The important differences.
- Which existing intuitions transfer safely.
- Where those intuitions become misleading.
- The shortest practice path from familiar competence to useful new competence.

Use at least one side-by-side example expressed in both domains. Completion requires a transfer check: give a new case in the target domain and ask the learner to solve it using the bridge.
